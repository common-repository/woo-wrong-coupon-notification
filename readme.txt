Get an email notification in the Admin User's email if someone uses wrong coupon code. [Helps to fix typos] 

== Description ==
Sometime you might have wrong coupon code published due to a typo, mistake of a copywriter or some other reason. It get frustrating for user when they try to use a coupon code but the store identify it as wrong code. This plugin sends you an email notification every time the wrong coupon is used. So, you can easily identify your mistakes and fix them.

== Installation ==
This is pure plug and play type plugin. You just need to install this and activate this. After that every time a wrong coupon is used, you'll get a email notification in site administrator's email address.
